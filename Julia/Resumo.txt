-----Funções-----

---parse = Converte uma string para um número (Int, Float etc).
num = parse(Int, readline())
num = parse(Float64, readline())

Conversão	Comando
String → Int	parse(Int, s)
String → Float	parse(Float64, s)
Int → Float	Float64(x)
Float → Int	Int(x) ou round(Int, x)
Qualquer coisa → String	string(x)
Vetor de Strings → Vetor de Int	parse.(Int, vetor)

---println = Significa print line, ou seja, imprime algo na tela e pula uma linha no final.
println("Olá Mundo!")

push = Adiciona um valor ao final de um vetor (lista).

---readline = Lê uma linha da entrada padrão 
O que vem de um readline é uma string
println("Digite seu nome:")
nome = readline()
println("Seja bem-vindo ", nome)

split = Divide uma string em partes usando um separador.
